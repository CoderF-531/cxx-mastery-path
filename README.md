# C++ Mastery Path

中文 | [English](#english)

## 中文

这是我的 C++ 工程师成长代码仓库，用于记录暑假高强度学习期间的每日代码、练习、项目和阶段性成果。

当前目标：

- 恢复并巩固 C++ 基础、面向对象、STL 和数据结构能力。
- 系统学习现代 C++、工程化、CMake、Git、调试和测试。
- 逐步补充 Linux、网络编程、MySQL、Redis 等 C++ 后端方向基础。
- 完成可展示项目，为实习、考研机试和项目面试做准备。

## 学习节奏

- 每天至少投入 5 小时。
- 每天完成代码后按时提交到本仓库。
- 每日产出应包含代码、运行结果、错误记录和复盘。
- 学习规划和验收记录维护在本地学习规划目录中。

## 推荐目录结构

```text
cxx-mastery-path/
├── diagnostics/        # 水平诊断代码
├── basics/             # C++ 基础复盘
├── data-structures/    # 数据结构练习
├── modern-cpp/         # 现代 C++ 练习
├── projects/           # 阶段项目
├── notes/              # 与代码强相关的简短笔记
└── tests/              # 测试代码
```

## 当前阶段

当前处于水平诊断与 C++ 基础恢复阶段。第一步是通过基础语法、类与对象、STL、数据结构练习确认真实水平，再决定后续补强顺序。

## 提交规范

建议提交信息格式：

```text
阶段: 完成内容
```

示例：

```text
diagnostics: add day 1 C++ baseline exercises
basics: practice constructors and copy control
modern-cpp: add unique_ptr ownership examples
```

- Rebuild and strengthen C++ fundamentals, object-oriented programming, STL, and data structures.
